# Лабораторна робота №1
## Тема: Знайомство з Vue.js 3 (Composition API)

Студент: Кравчук Максим  
Група: ВТ-22-1    

---

### Мета роботи
Вивчити основи роботи з фреймворком Vue.js 3, розробити односторінковий застосунок (SPA) з використанням компонентів, реактивних даних, механізмів рендерингу та локального збереження даних у LocalStorage.

---

### Завдання
Розробити ToDo застосунок, який дозволяє:
- додавати нові задачі
- редагувати задачі
- видаляти задачі
- змінювати статус задачі (active/done)
- фільтрувати задачі за статусом
- зберігати список задач у LocalStorage
- відображати кількість активних задач
- встановлювати пріоритет задачі (low/medium/high)
- зберігати дату створення задачі
- забезпечити базове оформлення інтерфейсу

---

### Використані технології
- Vue.js 3 (Composition API)
- Vite
- JavaScript (ES6+)
- LocalStorage API
- ESLint та Prettier
- CSS

---

### Інструкція з запуску
1. Встановити залежності:

npm install

2. Запустити застосунок у режимі розробки:

npm run dev

3. Відкрити застосунок у браузері за адресою:
http://localhost:5173/

---

### Опис структури проєкту

src/
├─ components/
│ └─ TodoList.vue – головний компонент списку задач
├─ App.vue – головний компонент застосунку
└─ main.js – точка входу

### Результати виконання
У результаті виконання лабораторної роботи був створений ToDo застосунок з підтримкою роботи з компонентами, реактивністю, фільтрацією задач, збереженням даних у LocalStorage та базовим оформленням.
