# Лабораторна робота №3
Тема: Стилізація компонентів у Vue.js

## Мета
Закріпити роботу зі стилями у Vue (scoped, CSS Modules), динамічними класами, застосуванням `<Teleport>` та анімаціями переходів.

## Опис проєкту
Міні-сайт **Vue News** з локальними даними новин:
- Головна сторінка зі списком новин
- Детальна сторінка новини `/article/:id`
- Інформаційна сторінка `/about`
- Перемикач теми (Light/Dark) зі збереженням у LocalStorage
- Рекламне модальне вікно на `<Teleport>`
- Анімація переходів між сторінками через `<Transition>`
- Обробка помилкових маршрутів (NotFound)

## Використані технології
- Vue 3 (Composition API)
- Vue Router 4
- Vite
- CSS Modules / scoped CSS

## Маршрути
- `/` — Головна сторінка
- `/about` — Про сайт
- `/article/:id` — Cторінка новини
- `/:pathMatch(.*)*` — NotFound (404)

## Запуск проєкту
Встановити залежності:
npm install

Запуск у режимі розробки:
npm run dev

## Структура
- `components` — UI-елементи (картки новин, хедер, футер, модалка)
- `views` — сторінки
- `router` — маршрутизація
- `data` — локальні дані
