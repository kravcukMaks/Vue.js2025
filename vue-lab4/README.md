# Лабораторна робота №4  
**Тема:** Vue 3, Pinia, VeeValidate, i18n  
**Студент:** Кравчук Максим  
**Група:** ВТ-22-1  

## Реалізований функціонал:
- Роутинг між сторінками: Profile, Products, Cart
- Локалізація (UA/EN)
- Збереження профілю користувача (Pinia + LocalStorage)
- Валідація даних формою (VeeValidate + Yup)
- Масив телефонів з можливістю додавання і видалення
- Список товарів
- Додавання товарів у кошик
- Підрахунок загальної суми та кількості
- Кошик відображається окремою сторінкою

## Використані технології:
- Vue 3 (Composition API)
- Pinia
- VeeValidate + Yup
- Vue Router
- Vue I18n
- TypeScript
- HTML/CSS

## Скриншоти роботи застосунку:
1. Профіль (валидація)
2. Список товарів
3. Кошик
