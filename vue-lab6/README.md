## Тема роботи
Nuxt.js 4 та Nuxt UI. Робота з файловим роутингом, серверними маршрутами Nitro,  
отриманням даних через useFetch, SEO-оптимізацією та бібліотекою компонентів Nuxt UI.

---

## Мета роботи
- Навчитись створювати Nuxt 4-проєкт та розуміти структуру директорій `app`, `pages`, `layouts`, `server`.
- Ознайомитись з файловим роутингом та динамічними маршрутами.
- Реалізувати серверні маршрути з використанням Nitro (`server/api`).
- Навчитись отримувати дані через `useFetch`.
- Використати SEO-інструменти Nuxt (`useSeoMeta`, sitemap, robots.txt).
- Використати базові компоненти бібліотеки Nuxt UI.

## Запуск проєкту

### Встановлення залежностей
```bash
npm install
```

### Запуск проєкту
```bash
npm run dev
```

```ts

useFetch(() => '/api/posts/' + route.params.slug)
```
